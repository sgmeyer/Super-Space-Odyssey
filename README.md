SuperSpaceOdyssey
=================

Html5 and JavaScript Game

Basic retro side scrolling shooter.  This reminds me of the olds days, but with slightly better art.

- Side scroller
- Shoot 'em up
- Phoenix style retro game

Play the game: http://sgmeyer.github.io/SuperSpaceOdyssey
